# Тесты на проверку получения данных о заказе по треку заказа в Яндекс.Самокат с помощью API Яндекс.Самокат.
- configuration.py - содержит URL стенда и эндпоинты API Яндекс.Самокат
- sender_stand_request.py - содержит функции для отправки HTTP-запросов к API сервиса: создание заказа и получение заказа по параметру "track"
- get_order_by_track_number_test.py - содержит автотест, который создаёт заказ, сохраняет его track, получает заказ по этому track и проверяет, что код ответа равен 200
- data.py - содержит тестовые данные, заголовки запросов и тело запроса в формате JSON
- Для запуска тестов должны быть установлены пакеты pytest и requests
- Запуск всех тестов выполняется командой python -m pytest -v
- database_tasks.sql - содержит SQL запросы, необходимые для решения задания 1 и 2
- autotest_run.jpg - скриншот выполнения проверки по составленному автотесту
